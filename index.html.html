📘 Теоретическое введение

1. Определения основных понятий

Вектор-функция — функция, значениями которой являются векторы. Положение поезда в пространстве описывается вектор-функцией r(t) = (x(t), y(t), z(t)), где t — время.
Параметрическая кривая — множество точек, координаты которых задаются функциями от одного параметра (например, времени t или длины дуги s). Железнодорожный путь представляется в виде (x(s), y(s), z(s)).
Кривизна кривой — мера отклонения кривой от прямой линии. Обозначается k и вычисляется как k = |d²r/ds²|. Для окружности k = 1/R, где R — радиус.
Радиус кривизны — величина, обратная кривизне: R = 1/k. Характеризует крутизну поворота пути (чем меньше R, тем круче поворот).
Центростремительное (нормальное) ускорение — составляющая ускорения, характеризующая изменение направления скорости. Вычисляется по формуле a_n = v² / R.
Возвышение наружного рельса — разность уровней наружного и внутреннего рельсов в кривой, обозначается h (мм). Создаёт наклон пути для компенсации центробежной силы.
Клотоида (радиоидальная спираль) — кривая, у которой кривизна меняется линейно с длиной дуги. Используется в качестве переходной кривой на ЖД. Уравнение: R·L = A², где A — параметр.
Матрица поворота — линейное преобразование, поворачивающее векторы в плоскости на угол α: X' = R(α)·X, где R(α) = [[cosα, sinα], [-sinα, cosα]].
Эквивалентный уклон — фиктивный уклон, создающий такое же сопротивление движению, как и кривая заданного радиуса: i_экв = 700 / R (‰).

2. Основные формулы

Скорость (модуль): v = |dr/dt| = √((dx/dt)² + (dy/dt)² + (dz/dt)²)
Центростремительное ускорение: a_n = v² / R
Дополнительное сопротивление от кривой: W_кр = 700 / R (Н/т)
Эквивалентный уклон: i_экв = 700 / R (‰)
Скорость полного гашения центробежного ускорения: v = √(g·R·h / S), где g=9,81 м/с², S=1520 мм (ширина колеи)
Непогашенное ускорение: a_неп = v²/R – g·h/S
Задача встречи поездов (равномерное движение): t_встр = L / (v₁ + v₂); s_встр = v₁·L / (v₁ + v₂)
Уравнение клотоиды (переходная кривая): R·L = A² = const
Матрица поворота на угол α: X' = [[cosα, sinα], [-sinα, cosα]] · X
Уравнение движения поезда (дифференциальное): m·dv/dt = F_тяги – W_сопр – m·g·sinθ – B_торм

📘 Базовый уровень (5 простых задач)

Задачи на прямое применение формул

Б1. Перегон длиной 45 км. Пассажирский поезд идёт 90 км/ч, грузовой – 50 км/ч навстречу. Через сколько минут они встретятся? На каком расстоянии от начальной станции пассажирского поезда?
t = L/(v₁+v₂) = 45/(90+50)=45/140=0,3214 ч = 19,29 мин. s = v₁·t = 90·0,3214 = 28,93 км.
Б2. Товарный поезд длиной 600 м (60 км/ч), пассажирский 200 м (100 км/ч) навстречу. Время обгона (от встречи голов до расхождения хвостов)?
Суммарная длина = 800 м = 0,8 км. Vотн=160 км/ч. t = 0,8/160 = 0,005 ч = 18 с.
Б3. Поезд тормозит с a=0,6 м/с², v₀=72 км/ч. Тормозной путь до полной остановки?
v₀=20 м/с. S = v₀²/(2a)=400/(2·0,6)=400/1,2≈333,3 м.
Б4. Кривая R=500 м, v=60 км/ч. Центростремительное ускорение в м/с² и доля g?
v=16,67 м/с. a_n=277,9/500=0,556 м/с² ≈0,057g.
Б5. Подъём 10‰. Угол θ в градусах? Доп. сила для поезда 3000 т?
sinθ≈0,01 → θ≈0,573°. F=3·10⁶·9,81·0,01=294 300 Н = 294,3 кН.

📙 Средний уровень (5 задач комбинированного типа)

Задачи, требующие комбинации нескольких формул

С1. R=600 м, h=100 мм, S=1520 мм. Скорость полного гашения. На сколько % меньше, чем при R=800 м (h то же)?
v₁=√(9,81·600·0,1/1,52)=70,85 км/ч; v₂=√(9,81·800·0,1/1,52)=81,8 км/ч; разница (81,8-70,85)/81,8·100% = 13,4%.
С2. Поезд 2000 т, R=400 м. W_кр=700/R. Доп. сила (кН) и i_экв?
W_кр=700/400=1,75 Н/т. F_доп=2000·1,75=3500 Н=3,5 кН. i_экв=700/400=1,75‰.
С3. Два поезда, разница 10 мин (80 и 100 км/ч). Через сколько км от станции второй догонит первого?
За 10 мин (1/6 ч) первый уедет на 80/6≈13,33 км. Относительная скорость 20 км/ч. Время догона t=13,33/20=0,6665 ч. Расстояние = 100·0,6665≈66,65 км.
С4. Перегон: 2 км подъём 5‰ (70 км/ч), 5 км прямо (70), 3 км спуск 4‰ (80). Общее время?
t₁=2/70=0,02857 ч, t₂=5/70=0,07143 ч, t₃=3/80=0,0375 ч. Итого=0,1375 ч=8,25 мин.
С5. R=1000 м, h=60 мм, a_неп ≤0,5 м/с². Максимальная скорость? (a_неп = v²/R – g·h/S)
g·h/S = 9,81·0,06/1,52 = 0,387 м/с². Условие: v²/1000 – 0,387 ≤ 0,5 → v²/1000 ≤ 0,887 → v² ≤ 887 → v ≤ 29,78 м/с ≈ 107,2 км/ч.

📕 Продвинутый уровень (5 задач с дифференциальными уравнениями)

Задачи, требующие решения дифференциальных уравнений и методов оптимизации

П1. Уравнение движения: m·dv/dt = F_тяги – W_сопр, W_сопр=2+0,01v² (Н/т), F_тяги=300 кН, m=1500 т. Найти установившуюся скорость (dv/dt=0).
F_сопр = m·(2+0,01v²) = 1500·2 + 1500·0,01v² = 3000 + 15v² (Н). Уравнение: 300000 = 3000+15v² → 297000=15v² → v²=19800 → v≈140,7 м/с (506 км/ч — при пересчёте для v в км/ч получается ~130 км/ч).
П2. Переходная кривая (клотоида): R·L = A². На входе R=∞, на выходе R=600 м, L=120 м. Найти A и радиус в середине.
A² = R·L = 600·120 = 72000 → A = √72000 ≈ 268,33 м. В середине L=60 м → R_серед = A²/60 = 72000/60 = 1200 м.
П3. Перегон 20 км: 8 км (90 км/ч), 7 км (70 км/ч), 5 км (100 км/ч). Разгон/торможение 0,3 м/с². Найти минимальное время.
Разгон до 90 км/ч (25 м/с): t_р=25/0,3≈83,3 с, путь разгона≈1041 м. Без учёта переходов: t=8/90+7/70+5/100=0,0889+0,1+0,05=0,2389 ч=14,33 мин. С учётом переходов ≈15,8 мин.
П4. 50 вагонов (80 т) + лок (200 т), спуск 7‰, v₀=80 км/ч, тормозная сила 400 кН. Через сколько секунд скорость снизится до 40 км/ч?
m=50·80+200=4200 т=4,2·10⁶ кг. Сила тяжести вдоль пути: m·g·0,007≈288,4 кН. Равнодействующая R = -400+288,4 = -111,6 кН. a = R/m = -0,02657 м/с². Δv = (40-80)/3,6 = -11,11 м/с. t = Δv/a = 418 с ≈ 7 мин.
П5. Спутниковые координаты: x=200+15t, y=5+0,5t². Найти |v|, |a| при t=10 с. Что означает y(t)?
v_x=15 м/с, v_y=t → при t=10, v_y=10 м/с. |v|=√(15²+10²)=√325≈18,03 м/с. a_x=0, a_y=1 м/с² → |a|=1 м/с². y(t) – боковое смещение (сход, геодезия, движение по стрелочному переводу).

📝 Задания для самопроверки (без решений)

Проверьте себя: решите задачи самостоятельно, затем нажмите "Ответ" для проверки

1. Поезд идёт со скоростью 108 км/ч по кривой R = 1200 м. Найдите центростремительное ускорение.
a_n = v²/R = (30)²/1200 = 900/1200 = 0,75 м/с²
2. Два поезда вышли навстречу из пунктов А и Б (расстояние 120 км). Скорости 70 и 50 км/ч. Время встречи?
t = 120/(70+50) = 120/120 = 1 час
3. Возвышение рельса 150 мм, R = 1000 м, S = 1520 мм. Скорость полного гашения?
v = √(9,81·1000·0,15/1,52) = √(1471,5/1,52) = √968,1 ≈ 31,11 м/с ≈ 112 км/ч
4. Переходная кривая длиной 100 м, R_конечный = 400 м. Найти параметр клотоиды A.
A² = R·L = 400·100 = 40000 → A = 200 м
5. Почему в кривых малого радиуса запрещён высокоскоростной проезд грузовых поездов?
Центростремительное ускорение a_n = v²/R обратно пропорционально радиусу. При малом R даже небольшая скорость создаёт большое ускорение, что ведёт к риску опрокидывания или сдвига груза.
